Dart诞生于2011年10月10日，是一种“结构化的Web编程”语言，应用于Web、服务器、移动应用等领域的开发。

1 概念

  • 概念
    • Dart程序有统一的程序入口，即main()函数入口，与Java、C或C++语言相似。
    • 一切皆对象，所有对象都是类的实例，所有对象都继承自内置的Object类，类似于Java语言。
    • Dart语言是弱数据类型，指定类型非必须，指定数据类型和编译时的常量，可提高运行速度。
    • Dart没有public、protected和private的概念，私有特性通过变量或函数加上下划线来表示。
    • Dart工具可以检查出warning警告信息和errors错误信息。
      • 警告信息表明代码可能不工作，但是不会妨碍程序运行。
      • 错误信息可能是编译时的错误，也可能是运行时的错误。
      • 编译时的错误将阻止程序运行，运行时的错误会以exception异常方式呈现。
// 入口方法
// main() {
//   print('Hello, Dart!');
// }

// 表示main方法没有返回值
void main() {
  // 打印信息
  print('Hello, Dart!');
}

// 注释方法一
/* 注释方法二 */
///注释方法三

  • 关键字

    • Dart支持anync/await异步处理，拥有56个关键字，具体如下。
    • abstract、as、library、switch、break、false、get、else、this、yield*、factory。
    • external、do、typeded、assert、super、await、try、enum、true、static、dynamic。
    • operator、is、extends、export、throw、mixin、new、case、void、async*、covariant。
    • deferred、in、default、return、const、class、for、null、while、rethrow、interface。
    • continue、if、finally、import、catch、final、var、part、with、sync*、set、implements。

  • 常用库

    • dart:svg：事件和动画的矢量图象支持。
    • dart.html：网页开发用到的库，使用频率高。
    • dart:math：数字常量及函数，提供随机数算法。
    • dart.io：文件读写I/O相关操作的库，使用频率高。
    • dart:async：异步编程支持，提供Future和Stream类。
    • dart:collection：对dart:core提供更多的集合支持。
    • dart:convert：不同类型(Json、UTF-8)间的字符编码、解码支持。
    • dart:core：Dart语言内建的类型、对象以及Dart语言核心的功能，使用频率高。
    • 使用任何库之前都需要加上import语句，官方提供的pub工具可以安装丰富的第三方库。

2 变量

  • 变量
    • Dart是脚本类语言，可以不预先定义变量类型，它会自动推导类型。
    • 可以使用var关键字来声明变量，也可以通过类型关键字来定义变量。
    • 命名规则
      • 变量名必须由数字、字母、下划线_和美元符$组成。
      • 标识符开头不能是数字，标识符不能是保留字和关键字。
      • 变量名区分大小写，尽量见名知义，变量名用名词，方法名用动词。
main() {
  // 定义变量使用var关键字
  var str1 = 'Hello, Dart!';
  print(str1);
  var num1 = 123;
  print(num1);

  // 通过类型关键字定义变量
  String str2 = 'Hello, Dart!';
  print(str2);
  int num2 = 12345;
  print(num2);
}
  • 常量
    • 若定义的变量不会变化，使用final或const关键字定义。
    • final是惰性初始化，即在运行时第一次使用前才初始化。
    • final的值只能被设定一次，是一个运行时常量，const则是一个编译时常量。
main() {
  String str = 'Hello, Dart!';
  // 变量可以改变值
  str = 'Hello, Golang!';
  print(str);

  // 常量不可以改变值
  // const关键字定义常量
  const PI1 = 3.1415;
  // 改变值将报错
  // PI1 = 3.14159267;
  print(PI1);

  // final关键字定义常量
  final PI2 = 3.14159;
  // PI2 = 3.141592535;
  print(PI2);

  // final是运行时常量
  final time1 = new DateTime.now();
  print(time1);
  // const是编译时常量
  // const time2 = new DateTime.now();
  // print(time2);
}

3 数据类型

  • 数据类型
    • 常用的数据类型
      • Number(数字)：int(整型)和double(浮点型)。
      • String(字符串)：String。
      • Boolean(布尔)：bool。
      • List(数组)：Dart中是列表对象，大多数人称之为列表。
      • Map(字典)：键值对相关的对象，键和值可以是任何类型的对象。
    • 用不到的数据类型
      • Runes：表示字符串中的UTF-32编码字符。
      • Symbol：表示在Dart程序中声明的运算符或标识符。
    • 可以使用is关键字来判断数据类型。
main() {
  // is关键字判断数据类型
  var str = '12345';
  // ignore: unnecessary_type_check
  if (str is String) {
    print('这是String类型。');
  } else if (str is int) {
    print('这是int类型。');
  } else {
    print('这是除String和int的其他类型。');
  }
}

3-1 Number

  • Number
    • int整型和double浮点型，都是Number类型的子类，int类型不能包含小数点。
    • 包含+、-、*、/、%以及位移操作等，常用的方法有abs、ceil和floor。
main() {
  // int，必须是整型
  int num1 = 123;
  print(num1);

  // double，既可以是整型，也可以是浮点型
  double num2 = 123;
  double num3 = 123.5;
  print(num2);
  print(num3);

  // 运算符
  var sum = num1 + num2;
  print(sum);
}

3-2 String

  • String
    • 字符串类型，使用单引号、双引号或三引号定义，开发中大量使用。
    • 通过两种方式进行字符串的拼接：使用$符号，或+符号进行操作。
main() {
  // 字符串定义的几种方式
  var str1 = 'This is string.';
  var str2 = "This is string.";
  print(str1);
  print(str2);

  String str3 = 'This is String.';
  String str4 = "This is String.";
  print(str3);
  print(str4);

  String str5 = '''
  This is String.
  This is String.
  ''';
  String str6 = """
  This is String.
  This is String.
  """;
  print(str5);
  print(str6);

  // 字符串的拼接
  String str7 = 'Hello';
  String str8 = 'Dart.';
  print('$str7, $str8');
  print(str7 + ', ' + str8);
}

3-3 Boolean

  • Boolean
    • 只有字面量true和false是布尔类型，这两个对象都是编译时常量。
    • Dart是强布尔类型检查，只有bool类型的值是true才被认为是true。
main() {
  // bool
  bool flag1 = true;
  print(flag1);
  bool flag2 = false;
  print(flag2);

  // 条件判断语句
  var flag3 = true;
  if (flag3) {
    print('这是真的。');
    // ignore: dead_code
  } else {
    print('这是假的。');
  }

  var a = 123;
  var b = 456;
  if (a == b) {
    print('a和b相等。');
  } else {
    print('a不等于b。');
  }
}

3-4 List

  • List
    • 具有一系列相同类型的数据，类似于JavaScript的Array数组对象。
    • List对象的第一个元素索引是0，最后一个元素索引是list.length-1。
main() {
  // List定义方式一
  var list1 = ['Lucy', 23, true];
  print(list1);
  // 打印List的长度和对应位置的值
  print(list1.length);
  print(list1[0]);

  // List定义方式二：指定类型
  var list2 = <int>[21, 23, 25];
  var list3 = <String>['Lucy', 'Lily'];
  print(list2);
  print(list3);

  // List定义方式三：增加数据
  var list4 = [];
  print(list4);
  print(list4.length);
  list4.add('Lily');
  list4.add('Lucy');
  list4.add(25);
  print(list4);
  print(list4.length);

  // List定义方式四：新版中已去除
  // var list5 = new List();
  // 创建一个固定长度的集合
  var list6 = List.filled(3, '');
  print(list6);
  list6[0] = 'Simon';
  list6[1] = 'Diana';
  list6[2] = 'Julius';
  print(list6);
  // 已固定长度，添加报错
  // list6.add('Monica');
  // print(list6);
  // 也无法通过修改长度进行改变
  // list6.length = 5;

  // 该定义方式可以改变集合长度
  var list7 = ['Damien', 'Monica'];
  list7.length = 0;
  print(list7);

  var list8 = List<String>.filled(3, '');
  list8[0] = 'String';
  list8[1] = 'Boolean';
  list8[2] = 'List';
  print(list8);
}

3-5 Map

  • Map
    • 将key和value关联在一起，也就是键值对，且key必须是唯一的。
    • 可以为Map对象添加新的键值对，通过length获取键值对的数量。
main() {
  // Map定义方式一
  var person1 = {'name': 'Lucy', 'age': 20, 'work': '程序员'};
  print(person1);
  print(person1['name']);

  // Map定义方式二：构造函数Map()
  var person2 = new Map();
  person2['name'] = 'Lily';
  person2['age'] = '26';
  person2['work'] = ['测试', 'UI设计师'];
  print(person2);

  // 添加新的键值对
  person2['sex'] = 'female';
  print(person2);

  // 使用length来获取键值对的数量
  print(person2.length);
}

4 运算符

  • 运算符

4-1 算术运算符

  • 算术运算符
    • +、-、*、/、%、~/(返回一个整数值的除法)、-expr(一元减号，也叫负号)。
    • 还支持前缀递增++var、前缀递减--var、后缀递增var++、后缀递减var--运算符。

4-2 关系运算符

  • 关系运算符
    • ==、!=、>、<、>=、<=。
    • 测试两个对象是否表示相同的事务，使用==运算符。
    • 确定两个对象是否完全相同，则用identical()函数。
main() {
  // 关系运算符
  int a = 5;
  int b = 3;
  print(a == b);
  print(a != b);
  print(a > b);
  print(a < b);
  print(a >= b);
  print(a <= b);

  // 用于条件判断
  if (a > b) {
    print('a大于b。');
  } else {
    print('a小于b。');
  }

  // 判断对象是否完全相同
  int c = 5;
  print(identical(a, c));
}
  • 类型测试操作符：as(类型转换)、is(对象是相应类型时返回true)、is!(对象不是相应类型时返回true)。
  • 赋值运算符
    • 使用=运算符赋值，当变量为null时使用??=运算符赋值。
    • 复合赋值运算符：+=、-=、*=、/=、%=、~/=、<<=、>>=、&=、^=、|=。
  • 逻辑运算符：!expr(反转表达式)、||(逻辑或)、&&(逻辑与)。
  • 位运算符：&(与)、|(或)、^(异或)、~expr(一元位补码)、<<(左移)、>>(右移)。
  • 条件表达式
    • expr1 ?? expr2：如果expr1为非空，返回其值，否则计算并返回expr2的值。
    • condition ? expr1 : expr2：如果条件为真，返回expr1，否则返回expr2(也叫三元表达式)。
  • 级联操作：使用两个点..表示，可对同一对象执行一系列操作。
    • 类似于JavaScript中Promise的then处理，目的是简化代码。
    • 严格来说级联的双点符号并非运算符，只是Dart语法的一部分。
main() {


  // 关系运算符


  // 类型测试操作符
  if (a is int) {
    print('a是int类型.');
  }

  // 赋值运算符
  // 赋值给value
  var value = a;
  print(value);
  // 如果z为空，则将值分配给z，否则z保持不变
  var z;
  z ??= value;
  print(z);

  // 逻辑运算符
  if (!false && (a == 1 || b == 6)) {
    print("逻辑运算符操作.");

  // 位运算符
  final values = 0x22;
  final bitmask = 0x0f;
  print(~values);
  print(values & bitmask);
  print(values & ~bitmask);
  print(values | bitmask);
  print(values ^ bitmask);
  print(values << 4);
  print(values >> 4);

  // 条件表达式
  var v = a > 1 ? b : c;
  print(v);
  String playerName(String name) => name ?? 'Lucy';
  print(playerName);

5 函数

  • 函数
    • Dart属于面向对象语言，函数属于Function对象，可像参数一样传递给其他函数，便于做回调处理。
    • 可选参数：将参数使用中括号[]括起来，用来表明是可选位置参数。
    • 参数默认值
      • 如果为参数指定了默认值，不传入值时，函数会使用该默认值。
      • 如果传入了值，则使用传入的值取代默认值，默认值通常为null。
    • 返回值特点：所有函数都有返回值，如果没有指定函数的返回值，则默认为null。
main() {
  // 获取用户信息
  String getUserInfo1(String name, String sex, [String? from]) {
    var info = '$name的性别是$sex.';
    if (from != null) {
      info = '$info来自$from.';
    }
    return info;
  }

  print(getUserInfo1('刘一', '男'));

  // 获取用户信息，使用等号来设置默认值参数
  String getUserInfo2(String name, String sex, [String? from = '中国']) {
    var info = '$name的性别是$sex.';
    if (from != null) {
      info = '$info来自$from.';
    }
    return info;
  }

  print(getUserInfo2('陈二', '男'));
}
